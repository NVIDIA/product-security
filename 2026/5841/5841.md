### Security Bulletin: NVIDIA Megatron Bridge - June 2026

Updated 2026-06-30

NVIDIA has released a software update for NVIDIA® Megatron Bridge.&nbsp;<div>To protect your system, clone or update this software to version 0.4.1 or later from the <a href="https://github.com/NVIDIA-NeMo/Megatron-Bridge">NVIDIA/Megatron-Bridge</a> GitHub repo.</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)
