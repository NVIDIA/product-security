### Details

This section provides a summary of potential vulnerabilities that this security update addresses and their impact. Descriptions use [CWE&trade;](https://cwe.mitre.org/), and base scores and vectors use [CVSS v3.1](https://www.first.org/cvss/specification-document) standards.

| **CVE ID** | **Description** | **Vector** | **Base Score** | **Severity** | **CWE** | **Impacts** |
| ---------- | ---------------- | ---------- | -------------- | ------------ | -------- | ------------ |
| CVE-2025-23351 | NVIDIA ConnectX and BlueField contain a vulnerability in the command interface where a local user with virtual function (VF) access may cause a write out of bounds by crafted input. A successful exploit of this vulnerability may lead to arbitrary code execution on the device. | [CVSS:3.1<br>/AV:A<br>/AC:L<br>/PR:L<br>/UI:N<br>/S:C<br>/C:H<br>/I:H<br>/A:H](https://www.first.org/cvss/calculator/3.1#CVSS:3.1/AV:A/AC:L/PR:L/UI:N/S:C/C:H/I:H/A:H) | 9.0 | CRITICAL | [CWE-787](https://cwe.mitre.org/data/definitions/787.html) | Code execution |
| CVE-2025-23350 | NVIDIA ConnectX and BlueField contain a vulnerability in the command interface where a local user with virtual function (VF) access may cause a write out of bounds by crafted input. A successful exploit of this vulnerability may lead to arbitrary code execution on the device. | [CVSS:3.1<br>/AV:A<br>/AC:L<br>/PR:L<br>/UI:N<br>/S:C<br>/C:H<br>/I:H<br>/A:H](https://www.first.org/cvss/calculator/3.1#CVSS:3.1/AV:A/AC:L/PR:L/UI:N/S:C/C:H/I:H/A:H) | 9.0 | CRITICAL | [CWE-787](https://cwe.mitre.org/data/definitions/787.html) | Code execution |

The NVIDIA risk assessment is based on an average of risk across a diverse set of installed systems and may not represent the true risk to your local installation. NVIDIA recommends evaluating the risk to your specific configuration.
