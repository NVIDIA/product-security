### Security Bulletin: NVIDIA TensorRT - July 2026

Updated 2026-07-14

NVIDIA has released a software update for NVIDIA® TensorRT.&nbsp;<div>To protect your system, clone or update this software to TensorRT v11.0.0 and Polygraphy 0.50.0 from the <a href="https://github.com/NVIDIA/TensorRT">NVIDIA/TensorRT</a> GitHub repo, <a href="https://developer.nvidia.com/tensorrt/download/11x">TensorRT product page</a>, the <a href="https://developer.download.nvidia.com/compute/cuda/repos/">network repository</a>, or Python package index (<a href="https://pypi.org/project/tensorrt/">TensorRT</a>, <a href="https://pypi.org/project/polygraphy/">Polygraphy</a>).</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)
