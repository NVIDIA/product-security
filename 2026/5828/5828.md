### Security Bulletin: NVIDIA Triton Inference Server - May 2026

Updated 2026-05-19

NVIDIA has released a software update for NVIDIA® Triton Inference Server.&nbsp;<div>To protect your system, clone or update this software to Triton Server r26.03 or later from the NVIDIA <a href="https://github.com/triton-inference-server/server">Triton Inference Server</a> GitHub repo.</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)
