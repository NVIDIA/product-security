### Security Bulletin: NVIDIA NeMo Framework - February 2026

Updated 2026-02-17

NVIDIA has released a software update for NVIDIA® NeMo Framework.&nbsp;<div>To protect your system, clone or update this software to version 2.6.1 or later from NVIDIA/NeMo Framework on <a href="https://github.com/NVIDIA/NeMo/releases">NVIDIA Github</a> or <a href="https://pypi.org/project/nemo-toolkit/">pypi</a>.</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)
