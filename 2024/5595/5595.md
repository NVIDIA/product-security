### Security Bulletin: NVIDIA Base Command Manager - November 2024

Updated 2024-11-19

NVIDIA has released a software update for NVIDIA Base Command Manager to address the security issues listed below.<div>To protect your system, download and install this software update through the BCM Package Repository.</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)

_______________________________________________________________________________________________________________________________________________

### Details

| **CVE ID** | **Description** | **Vector** | **Base Score** | **Severity** | **CWE** | **Impacts** |

### Notes

NVIDIA Base Command Manager 10.24.07 and earlier versions do not contain this vulnerability.<div>Use this security update to resolve the issue as follows:</div><div><ul><li>Update to the most recent version of CMdaemon on the head nodes and in all software images.</li><li>Update the nodes by rebooting them or by resynchronizing them against the software image.</li></ul></div>
