### Security Bulletin: NVIDIA Base Command and Bright Cluster Managers - November 2024

Updated 2024-11-26

NVIDIA has released a software update for NVIDIA Base Command and Bright Cluster Managers to address the security issues listed below.<div>To protect your system, update the CMdaemon package on the head node and in software images.</div>

Go to [NVIDIA Product Security](https://www.nvidia.com/security/)

_______________________________________________________________________________________________________________________________________________

### Details

This section provides a summary of potential vulnerabilities that this security update addresses and their impact. Descriptions use [CWE™](https://cwe.mitre.org/), and base scores and vectors use [CVSS v3.1](https://www.first.org/cvss/specification-document) standards.

| **CVE ID** | **Description** | **Vector** | **Base Score** | **Severity** | **CWE** | **Impacts** |
| ---------- | ---------------- | ---------- | -------------- | ------------ | -------- | ------------ |
| CVE-2024-0139 | NVIDIA Base Command Manager and Bright Cluster Manager for Linux contain an insecure temporary file vulnerability. A successful exploit of this vulnerability might lead to denial of service. | [CVSS:3.1/AV:L/AC:H/PR:L/UI:R/S:U/C:N/I:N/A:H](https://www.first.org/cvss/calculator/3.1#CVSS:3.1/AV:L/AC:H/PR:L/UI:R/S:U/C:N/I:N/A:H) | 4.4 | MEDIUM | [CWE-377](https://cwe.mitre.org/data/definitions/377.html) | Denial of Service |

The NVIDIA risk assessment is based on an average of risk across a diverse set of installed systems and may not represent the true risk to your local installation. NVIDIA recommends evaluating the risk to your specific configuration.

### Security Updates

The following table lists the software products and versions affected, and the updated version available from nvidia.com that includes this security update.

| **CVE IDs Addressed** | **Affected Products** | **Platform or OS** | **Affected Versions** | **Updated Version** |
| --------------------- | --------------------- | ----------------- | --------------------- | ------------------- |
| CVE-2024-0139 | NVIDIA Base Command Manager | Linux | Base Command Manager 3, Base Command Manager 10 | 10.24.09a |
| CVE-2024-0139 | NVIDIA Bright Cluster Manager | Linux | Bright Cluster Manager 9.0-9.2 | Bright Cluster Manager 9.0-22, 9.1-19, 9.2-17 |

### Notes

Use this security update to resolve the issue as follows:<div><ul><li>Update to the most recent version of CMdaemon on the head nodes and in all software images.</li><li>Update the nodes by rebooting them or by resynchronizing them against the software image.</li></ul></div>
